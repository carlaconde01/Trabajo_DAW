Etiqueta header

Para definir un bloque de contenido que hara las veces de titulo de la pagina web

Etiqueta nav

Aqui podriamos insertar enlaces al resto de nuestra pagina, para que el usuario

pueda desplazarse entre las partes de nuestro sitio web

Etiqueta section

Etiqueta que sirve para definir grandes zonas en nuestra pagina

Etiqueta article

Esta es mi primera pagina completa

Marca los limites de un contenido especifico, como un articulo o como una entrada en un blog

Etiqueta aside

Sirve para definir un contenido que esta relacionado con la pagina,
pero que se debe considerar como separado del contenido original

Etiqueta footer

Zona del pie de pagina
normalmente el nombre del autor y la fecha de publicacion
